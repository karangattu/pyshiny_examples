This app demonstrates multiple ways to use `ui.card_header()` with various parameters and configurations:

1. Basic header with plain text
2. Header with HTML formatting
3. Header with an icon
4. Header with custom container and CSS classes
5. Header with nested elements (like an action button)
6. Dynamic header that changes on button click
7. Header with a tooltip
8. Header with Font Awesome icons

Key features demonstrated:
- Use of `ui.card_header()` parameters
- Interactive elements
- Dynamic content generation
- Responsive layout with `ui.layout_columns()`
- Reactive effects
- Data visualization
- Tooltips

To run this app, ensure you have the following dependencies:
- shiny
- pandas
- numpy
- matplotlib

The app showcases the flexibility of `ui.card_header()` and provides a comprehensive example of its usage in Shiny for Python's express mode.
