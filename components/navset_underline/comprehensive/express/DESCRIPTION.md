This Shiny for Python app demonstrates the following parameters and features of `navset_underline`:

1. `id`: Provides a unique identifier for tracking the selected navigation tab
2. `selected`: Sets the initially selected tab
3. `header`: Adds a markdown header to the navigation set
4. `footer`: Adds a footer with additional information

Additional features:
- Uses synthetic data for sales and product information
- Includes multiple navigation panels
- Demonstrates `nav_menu` for additional navigation options
- Uses `layout_columns` and `card` for structured presentation
- Renders plots and data tables
- Shows the currently selected tab

The app showcases a comprehensive use of Shiny for Python's navigation and rendering capabilities while highlighting the flexibility of the `navset_underline` component.
