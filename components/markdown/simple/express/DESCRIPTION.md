This app demonstrates several key features:

1. Use of `ui.markdown()` to create rich, formatted text
2. Synthetic data generation without external files
3. Dynamic markdown content generation based on user selection
4. Express mode Shiny for Python implementation

The app provides:
- A sidebar for region selection
- A static markdown introduction
- Dynamic markdown content that updates based on the selected region
- Emoji and formatting to make the markdown more engaging

Key points from the function reference documentation used:
- `ui.markdown()` can accept multi-line strings
- Markdown supports headers, lists, and inline formatting
- Can be used both statically and dynamically

To run the app, save this script and execute it with a Shiny for Python runtime.
